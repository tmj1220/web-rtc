# Rokid-RTC-JS-SDK

## 文件目录


- example 演示版内容
- src
  - index.ts SDK入口文件
- rollup.config.ts rollup打包配置
- tsconfig.json ts配置文件
- tsconfig.eslint.json ts-eslint配置
- webpack.config.js webpack本地demo配置文件
- version.config.json 版本控制文件

## 打包

```shell
npm run build-SDK
tar -czvf rokid-rtc-sdk-[version].tgz lib
```
